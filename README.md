# Football Supporter Carpooling App

A complete full-stack carpooling application for football supporters.

## Tech Stack
- Backend: Node.js + Express + TypeScript + MongoDB
- Frontend: React + Vite + TypeScript + TailwindCSS

## Setup Instructions

### Backend
```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
